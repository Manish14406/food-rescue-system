🍽️ FoodRescue – Food Waste Management System
📌 Overview

FoodRescue is a web-based platform designed to reduce food waste and address food insecurity by connecting food donors with recipient organizations. The system allows restaurants, hotels, and event organizers to donate surplus food, while NGOs and shelters can browse, claim, and track food deliveries in real time.

The project demonstrates a complete full-stack application using modern web technologies and provides a structured workflow for food redistribution.
